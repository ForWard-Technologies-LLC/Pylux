# PS4 and PS5 Remote Play on Windows with Pylux

Pylux provides PS4 and PS5 Remote Play for Windows as an installer and portable zip, available on the GitHub Releases page.

## Download

[Download for Windows](https://github.com/ForWard-Technologies-LLC/Pylux/releases){ .md-button .md-button--primary target="_blank" rel="noopener" }

- **Installer** — recommended for most users
- **Portable zip** — unzip and run, no installation required

## Setup

1. Download and run the installer (or unzip the portable build)
2. Follow the [Setup Overview](index.md) to register your PS4 or PS5 console
3. Configure [Remote Connection](remoteconnection.md) to connect over the internet (Internet Play)

## Features on Windows

- Internet Play — connect to your PS5 or PS4 from anywhere
- Automatic console discovery on your local network
- Controller support
- No app-store required — direct download
