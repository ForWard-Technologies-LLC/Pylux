# PS5 Remote Play on Steam Deck with Pylux

Pylux runs on Steam Deck via Linux builds (AppImage, portable zip, or Flatpak). It supports Game Mode integration with custom controller profiles and automation scripts.

## Download

[View Linux / Steam Deck Releases](https://github.com/ForWard-Technologies-LLC/Pylux/releases){ .md-button .md-button--primary target="_blank" rel="noopener" }

Available as:

- **Flatpak** — from Flathub (`io.github.ForWard_Technologies_LLC.Pylux`)
- **AppImage** — make executable and run directly
- **Portable zip** — unzip and run `launch.sh`

## Setup

1. Download and install Pylux using one of the options above (see [Installation](installation.md) for detailed steps)
2. Switch to **Desktop Mode** on your Steam Deck
3. Follow the [Setup Overview](index.md) to register your PS5 or PS4 console
4. Set up [Automation](automation.md) to launch Pylux directly from Game Mode
5. Configure your [Controller Profile](controlling.md) for Steam Deck controls

## Features on Steam Deck

- Game Mode integration — launch directly from your Steam library
- Automation scripts — wake and connect to your console automatically
- Custom Steam Deck controller profile
- Internet Play — connect from anywhere without port forwarding
- HDR support on Steam Deck OLED (PS5 only)
