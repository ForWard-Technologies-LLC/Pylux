# Pylux Releases

All Pylux releases are published on the [GitHub Releases page](https://github.com/ForWard-Technologies-LLC/Pylux/releases){ target="_blank" rel="noopener" }.

Each release includes:

- Download links for all platforms (Android, iOS, macOS, Windows, Linux, AppImage, Flatpak)
- Release notes with changes and fixes
- Direct links to Google Play, the App Store, and the Mac App Store

## Updating

=== "Android / Android TV"
    Updates are delivered automatically through the Google Play Store.

=== "iOS / macOS"
    Updates are delivered automatically through the App Store / Mac App Store.

=== "Linux Flatpak"
    ```bash
    flatpak update io.github.ForWard_Technologies_LLC.Pylux
    ```

=== "Windows / Linux AppImage / Portable"
    Download the latest build from the [Releases page](https://github.com/ForWard-Technologies-LLC/Pylux/releases){ target="_blank" rel="noopener" } and replace your existing installation.
